### WeUI

WeUI是一套同微信原生视觉体验一致的基础样式库，由微信官方设计团队为微信 Web 开发量身设计，可以令用户的使用感知更加统一。包含`button`、`cell`、`dialog`、 `progress`、`toast`、`article`、`icon`等各式组件。

[https://github.com/weui/weui](https://github.com/weui/weui)

### Angular2-WeUI

使用`angular2`的方式来构建界面
